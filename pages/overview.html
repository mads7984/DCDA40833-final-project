Data Overview

Full dataset for selected Type 2 diabetes medications across four drug coupon sites.

Overview of the Dataset

This page presents the full dataset used in the project. The tables below show how each selected Type 2 diabetes medication is listed across TrumpRx, GoodRx, SingleCare, and Cost Plus Drugs.

Each table includes whether the drug is listed, the listed price, price type, dosage, quantity, transparency score, and a direct link to the platform page when available.

Dataset by Drug

The dataset is organized by drug to make it easier to compare how each platform presents the same medication.
